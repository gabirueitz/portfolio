Habilidades

Explore o átomo para saber mais sobre minhas habilidades:

HTML

É uma linguagem de marcação utilizada para criar e estruturar páginas da web. Usando tags para marcar elementos como cabeçalhos, parágrafos, links, imagens e tabelas define o conteúdo e layout de páginas da web. Faz parte da "tríade do FrontEnd".

CSS

É uma linguagem de estilo usada para controlar a aparência e o layout de páginas da web. Permite definir cores, fontes, espaçamentos, posicionamento e outros estilos para elementos HTML, melhorando o design e a experiência do usuário. Também faz parte da "tríade do FrontEnd".

JavaScript

É uma linguagem de programação de alto nível, usada para criar interatividade e dinamismo em páginas da web. Permite manipular elementos HTML, responder a eventos, validar formulários, criar animações e desenvolver aplicativos web avançados. Último componente da "tríade do FrontEnd".

Git

É um sistema de controle de versão distribuído, amplamente utilizado no desenvolvimento de software. Permite gerenciar alterações em projetos, rastreando o histórico de modificações, facilitando o trabalho em equipe, o controle de versões e a colaboração eficiente.

VSCode

VSCode (Visual Studio Code) é um editor de código-fonte altamente popular e amplamente utilizado. Desenvolvido pela Microsoft, oferece uma interface leve e poderosa, com recursos avançados de edição, depuração e integração com diversas linguagens de programação e extensões.

NodeJS

É um ambiente de execução de código JavaScript fora do navegador. Ele permite que os desenvolvedores criem aplicativos de servidor altamente escaláveis e em tempo real, usando JavaScript tanto no lado do cliente quanto no lado do servidor. Muito útil para o desenvolvimento BackEnd.

Python

É uma linguagem de programação de alto nível, interpretada e de propósito geral. Com uma sintaxe simples e legível, é amplamente utilizada para desenvolvimento de software, automação, análise de dados e criação de aplicações web.
